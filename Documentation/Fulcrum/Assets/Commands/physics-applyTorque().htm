void FulcrumPhy::applyTorque ( int id, float X, float Y, float Z )

Apply torque defined in the global co-ordinate frame of an actor, you can think of torque as being a rotational force about a given axis.

Parameters:
int id - Object id.
float X - Magnitude of torque about the x-axis.
float Y - Magnitude of torque about the y-axis.
float Z - Magnitude of torque about the z-axis.

Returns:
