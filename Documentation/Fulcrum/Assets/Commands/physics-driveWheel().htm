void FulcrumPhy::driveWheel ( int vehicleID, int wheelID, float torque )

This command will drive your wheel, you can apply it to any amount or combination of wheels on your vehicle.

Parameters:
int vehicleID - ID of the vehicle actor/object that the wheel belongs to
int wheelID - ID that you give the wheel upon creation
float torque - Amount of angular force to apply to the wheel


Returns:
