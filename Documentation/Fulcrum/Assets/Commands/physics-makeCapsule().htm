bool FulcrumPhy::makeCapsule ( int id, bool dynamic, float density, float radius, float length )

Creates a collision/physics capsule around a given object.

Parameters:
int id = Object id
bool dynamic - Set to TRUE to make the actor dynamic, FALSE for static.
float density - Sets the density of the actor, this only has any effect if the actor is dynamic.
float radius - Sets the radius of capsule.
float length - Sets the length of capsule.

Returns:
True if succeeds, false otherwise.
