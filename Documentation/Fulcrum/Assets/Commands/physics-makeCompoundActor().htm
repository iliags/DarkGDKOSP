bool FulcrumPhy::makeCompoundActor ( int id, bool dynamic, bool density)

You first need to make some shapes using makeBoxShape(), makeSphereShape() or makeConvexShape(). These shapes go onto a list, when you call this function it uses these shapes to create an actor with multiple shapes, the list is then cleared. The centre of the created actor will be (0, 0, 0), the real world position of each shape becomes the local position of the shape inside the new actor.

Parameters:
int id - Actor ID.
bool dynamic - Set to TRUE to make actor dynamic, FALSE for static.
float density - Density value of actor.

Returns:
TRUE if succeeds, FALSE otherwise.
