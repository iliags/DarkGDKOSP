bool FulcrumPhy::makeControllerCapsule ( int id, float maxStep, float slopeLimit, bool yOffset )

Creates a character controller, this is a special kinematic actor set up to give twitch control to your in-game characters as well as the ability to walk up steps and slopes.

Parameters:
int id - Object id.
float maxStep - Maximum height the controller can climb over.
float slopeLimit - Maximum angle the controller can walk up, in degrees.
bool yOffset - Set to true if the character models feet are at local position(0, 0, 0).

Returns:
TRUE if succeeds, FALSE otherwise.




bool FulcrumPhy::makeControllerCapsule ( int id, float maxStep, float slopeLimit, float radius, float height )

Creates a character controller, this is a special kinematic actor set up to give twitch control to your in-game characters as well as the ability to walk up steps and slopes. This overload lets you set the initial dimensions manually.

Parameters:
int id - Object id.
float maxStep - Maximum height the controller can climb over.
float slopeLimit - Maximum angle the controller can walk up, in degrees.
float radius - Radius of controller.
float height - Height of the controller.

Returns:
TRUE if succeeds, FALSE otherwise.
