bool FulcumPhy::makeConvexMesh ( int id, bool dynamic, float density)

Creates a convex mesh, this is usually dynamic and is used for more complex objects with a limit of 256 polygons, the object should be of a convex shape.

Parameters:
int id - Controller id.
float dynamic - Set to TRUE to create dynamic actor, FALSE for static.
float density - Density value of actor.

Returns:
TRUE if succeeds, FALSE otherwise.
