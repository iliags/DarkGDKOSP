bool FulcrumPhy::makeConvexShape ( int id )

This creates a convex shape, this will be put on the shape list. You can create many shapes and then call makeCompoundActor() to create an actor with the multiple shapes from the list.

Parameters:
int id - id of the object the shape will be based on, there is usually no reason to keep the object after you create the shape, its used more as a visual aid.


Returns:
TRUE if succeeds, FALSE otherwise.
