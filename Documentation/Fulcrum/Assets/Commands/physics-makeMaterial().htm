void FulcrumPhy::makeMaterial ( int materialID, float resitution, float dynamicFriction, float staticFriction )

Makes a new material which you may apply to actors/objects using FulcrumPhy::setMaterial(). You can give your actors/objects different properties to simulate the real world material they are supposed to be made of, eg metal, wood, rubber etc.

Parameters:
int materialID - ID that will be used to reference this material.
float restitution - Bounciness of the material, usually a value between 0-1.
float dynamicFriction - Dynamic friction of the material, usually a value between 0-1.
float staticFriction - Static friction of the material, usually a value between 0-1.

Returns:
