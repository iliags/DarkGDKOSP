void FulcrumPhy::positionCharacterController ( int id, float posX, float posY, float posZ )

This is a teleport function, it does not check for collisions, can be used for repositioning your character when they respawn, teleport etc.

Parameters:

int id - Controller id.
float posX - New controller position along the x-axis.
float posY - New controller position along the y-axis.
float posZ - New controller position along the z-axis.

Returns:
