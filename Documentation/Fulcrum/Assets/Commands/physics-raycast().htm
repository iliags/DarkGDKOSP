int FulcrumPhy::raycast ( float originX, float originY, float originZ, float directionX, float directionY, float directionZ )

Casts a ray from given position along given direction, will return ID of any object/actor it hits. You can get more information from the hit by using the other raycast commands, these commands will always return the results from the last ray cast.

Parameters:
float originX - Position on the x-axis that ray will start from
float originY - Position on the y-axis that ray will start from
float originZ - Position on the z-axis that ray will start from
float directionX - Angle about the x-axis for ray to travel
float directionY - Angle about the y-axis for ray to travel
float directionZ - Angle about the z-axis for ray to travel

Returns:
int - ID of first object/actor the ray hits
