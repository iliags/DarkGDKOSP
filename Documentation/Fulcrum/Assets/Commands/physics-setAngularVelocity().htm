void FulcrumPhy::setAngularVelocity ( int id, float X, float Y, float Z )

Use this to set your dynamic(non-kinematic) actors angular velocity manually.
Use this command sparingly as dynamic(non-kinematic) actors angular velocity should be controlled by applying forces, this is useful for re-setting a scene etc.

Parameters:
int id - Object/Actor id.
float X - New velocity about the x-axis.
float Y - New velocity about the y-axis.
float Z - New velocity about the z-axis.

Returns:
