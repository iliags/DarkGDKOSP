void FulcrumPhy::setCollisionGroup ( int id, int group )

Places actor/object in a particular collision group, you can use FulcrumPhy::setGroupCollision() to set which groups collide/interact with which. You can use these commands to save the physics solver from checking for collisions which are not possible or desired, therefore saving cpu.

Parameters:
int id - Object/Actor id
int group - Collision group

Returns:
