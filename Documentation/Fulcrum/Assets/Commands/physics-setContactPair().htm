void FulcrumPhy::setContactPair ( int objectA, int objectB )

This command will setup two actors/objects as a contact pair, if they come into contact, the contact will be recorded and placed on a stack, you can check for contacts on the stack using fulcrumPhy::getContactData().

Parameters:
int objectA - An actor/object to be paired
int objectB - An actor/object to be paired

Returns:
