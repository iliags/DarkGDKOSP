bool FulcrumPhy::setControllerDimensions ( int id, float increaseX, float increaseY, float increaseZ, bool collisionCheck )

If you find that your box controller dimensions don't fit your object use this function to change them. Note that controllers do not rotate so the x and z values should be the same for most applications.

Parameters:
int id - Controller id.
float increaseX - Increase in controller size along the x-axis.
float increaseY - Increase in controller size along the y-axis.
float increaseZ - Increase in controller size along the z-axis.
bool collisionCheck - if set to TRUE, the controller will only change size if there is enough space around it.

Returns:
True if controller changed size.
