void FulcrumPhysics::setControllerLocalPosition ( bool useDebugger, float yOffset )

Sets a character controllers local position, you may find that sometimes your models feet may come to rest above or below the ground, use this command to adjust.
Note: Be aware that character controllers are created with the default skin width, this means that the controller actor sits above the ground by this amount.

Parameters:
int id - Controller id.
float yOffset - Offset along the y-axis.

Returns:
