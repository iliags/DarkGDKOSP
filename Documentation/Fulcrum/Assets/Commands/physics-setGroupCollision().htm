void FulcrumPhy::setGroupCollision ( int groupA, int groupB, bool collide )

This command lets you determine whether two groups should collide with each-other or not. You can place actors/objects into groups using FulcrumPhy::setCollisionGroup(). You can use these commands to save the physics solver from checking for collisions which are not possible or desired, therefore saving cpu.

Parameters:
int groupA - A collision group
int groupB - A collision group
bool collide - Set to FALSE to disable collisions between groups, TRUE to enable collisions

Returns:
