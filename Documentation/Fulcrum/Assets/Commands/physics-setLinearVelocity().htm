void FulcrumPhy::setLinearVelocity ( int id, float X, float Y, float Z )

Use this to set your dynamic(non-kinematic) actors velocity manually.
Use this command sparingly as dynamic(non-kinematic) actors velocity should be controlled by applying forces, this is useful for re-setting a scene etc.

Parameters:
int id - Object/Actor id.
float X - New velocity along the x-axis.
float Y - New velocity along the y-axis.
float Z - New velocity along the z-axis.

Returns:
