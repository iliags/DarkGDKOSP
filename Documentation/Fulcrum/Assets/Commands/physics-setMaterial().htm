void FulcrumPhy::setMaterial ( int actorID, int materialID )

This sets your actor/object to the given material, this gives your actors/objects different properties to simulate the real world material they are supposed to be made of, eg metal, wood, rubber etc.

Parameters:
int actorID - Actor/Object you want to apply the material to.
int materialID - ID of the material to be used.

Returns:





void FulcrumPhy::setMaterial ( int vehicleID, int wheelID, materialID )

This sets the given wheel to the given material, this gives your wheels different properties to simulate the real world material they are supposed to be made of, eg rubber, wood etc.

Parameters:
int vehicleID - ID of the wheels owning vehicle.
int wheelID - ID you give the wheel upon creation.
int materialID - ID of the material to be used.

Returns:
