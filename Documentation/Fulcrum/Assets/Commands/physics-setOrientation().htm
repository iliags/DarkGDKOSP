void FulcrumPhy::setOrientation ( int id, float X, float Y, float Z )

Use this to rotate your dynamic(non-kinematic) or static actors manually.
For dynamic actors, use this command sparingly as dynamic(non-kinematic) actors must be rotated by applying forces, this is useful for re-setting a scene etc.
For static actors, also use this command sparingly or not at all if possible, if you do use it you will also need to rotate your rendered geometry to match. Character controllers need to be informed if the static geometry changes due to the way they process collision data.

Parameters:
int id - Actors id.
float X - New rotation about the x-axis.
float Y - New roatation about the y-axis.
float Z - New rotation about the z-axis.

Returns:
