void FulcrumPhy::setPosition ( int id, float posX, float posY, float posZ )

Use this to re-position your dynamic(non-kinematic) or static actors manually.
For dynamic actors, use this command sparingly as dynamic(non-kinematic) actors should be moved by applying forces, this command is useful for re-setting a scene or teleporting etc.
For static actors, also use this command sparingly or not at all if possible, if you do use it then you will also have to re-position your rendered geometry to suit. Character controllers need to be informed if the static geometry changes due to the way they process collision data.

Parameters:
int id - Object/Actor id.
float posX - New position along the x-axis.
float posY - New position along the y-axis.
float posZ - New position along the z-axis.

Returns:
