void FulcrumPhy::setWheelSuspensionTravelMax ( int vehicleID, int wheelID, float distance )

This command will set the maximum distance the wheel can travel vertically away from the vehicle as the vehicle comes away from the ground. This simulates vehicle suspension, a soft contact is made with the surface if the car raises off the ground less than the maximum suspension travel distance, a hard contact is made when the wheel is in its initial local position, in other words the suspension travel is at zero.

Parameters:
int vehicleID - ID of the vehicle actor/object that the wheels belongs to
int wheelID - ID you give the wheel upon creation
float distance - Maximum distance the wheel can travel vertically


Returns:
