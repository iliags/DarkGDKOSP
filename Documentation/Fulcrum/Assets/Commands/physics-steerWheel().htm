void FulcrumPhy::steerWheel ( int vehicleID, int wheelID, float angle )

This command will turn your wheel, you can apply it to any amount or combination of wheels on your vehicle.

Parameters:
int vehicleID - ID of the vehicle actor/object that the wheel belongs to
int wheelID - ID that you give the wheel upon creation
float torque - Angle, in degrees, to turn the wheel
