bool FulcrumPhy::swapObject ( int actorID, int newObjectID )

This will change the object that the dynamic actor updates, after use the actor(actorId) is re-assigned to the new object(newObjectId).

Parameters:
int actorID - ID of the actor you wish to alter.
int newObjectID - ID of the object you want the actor to represent, remember the actor now has the new ID too.

Returns:
TRUE if succeeds, FALSE otherwise.
