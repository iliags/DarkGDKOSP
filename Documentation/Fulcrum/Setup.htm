SETUP



Create a Visual Studio project using one of the DarkGDK templates provided when you installed the GDK, or you can use an existing project if you like.
Go to: Project -> “project name” Properties -> Configuration properties-> C/C++ ->Code generation Change the runtime library from “Multi-threaded Debug (/Mtd)” to Multi-threaded (/MT).


Header Files

In the Physics plug-in (version) folder there should be an include file, this is all the include files in one folder for ease of use, you can include this into all your future projects by going to: Tools -> options… -> Projects and Solutions -> VC++ Directories Select “Include files”, click on the new line folder and copy the “Include” address into here, it is usually something like this: C:\Users\Matty\Documents\Fulcrum Physics plug-in (version)\ Include
Alternatively you can empty all the contents of the Include file into one of the includes that are already in VC++ Directories, usually something like this: C:\Program Files\The Game Creators\Dark GDK\Include. I prefer the first method to keep everything separate.

Library Files

Library Files You need to do the same with the Lib folder that is included in the Fulcrum Physics plug-in (version) folder provided: Tools -> options… -> Projects and Solutions -> VC++ Directories Select “Library files”, click on the new line folder and copy the “Lib” address into here.
Again, as an alternative you can empty all the contents of the lib folder into an existing VC++ Directory such as: C:\Program Files\The Game Creators\Dark GDK\Lib\VS9

Dll's

You will need to put the contents of the dlls folder Fulcrum Physics plug-in (version) into the root directory of your project, this will be the folder where your “main.cpp” file is located.

Drivers

You may also need to update your drivers HERE, you may not though, so you may want to leave this until you know if you need it.



O.K, you should be good to go, don’t forget to put:
#include “FulcrumPhy.h”
at the top of your code.
Good luck.
