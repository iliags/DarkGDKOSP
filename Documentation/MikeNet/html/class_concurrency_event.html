Initializes object.

Parameters:
initialState True if the event object should start signaled, false if it should start unsignaled.
manualReset When the state of a manual-reset event object is signaled, it remains signaled until it is explicitly reset to nonsignaled by the ConcurrencyEvent::Set function.

When the state of an auto-reset (non manual-reset) event object is signaled, it remains signaled until a single waiting thread is released (e.g. ConcurrencyEvent::WaitUntilSignaled returns); the system then automatically resets the state to nonsignaled. If no threads are waiting, the event object's state remains signaled.
