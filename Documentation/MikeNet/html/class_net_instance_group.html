Detailed Description

Manages a group of instances.

Remarks:
Michael Pryor, 6/5/2010.

This instance is used for managing a collection of instances. These instances can be identified by ID (unique only within this instance group).

This class is thread safe; however, instances can only be 'finished' by the main process.
