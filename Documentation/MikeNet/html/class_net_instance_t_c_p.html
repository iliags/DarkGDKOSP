Detailed Description

TCP instance class responsible for managing all TCP operations on at an instance level.

Remarks:
Michael Pryor, 6/28/2010.

This class should be inherited by any instance class wishing to make use of Transmission Control Protocol.

virtual void NetInstanceTCP::FlushRecvTCP ( size_t  clientID  )  [pure virtual]

Empties the received TCP packet queue.

When TCP packets are received and a receive function has not been set (in the constructor), packets are put into a queue and received using GetPacketFromStoreTCP. If the queue gets too big newer packets may not be received for some time and so it can be useful to empty the queue. This command discards everything in the queue and has no impact on the partial packet store.

Parameters:
clientID ID of client to flush.
