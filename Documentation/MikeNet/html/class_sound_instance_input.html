void SoundInstanceInput::DealWithData ( WAVEHDR *  buffer  )  [virtual]

Called indirectly by callback thread to deal with a completed buffer.

The callback thread must not directly call this method, but instead posts a message to a thread initiated by this API. This restriction is mentioned in the waveform documentation on MSDN.

If unpaused the buffer will be re-queued to receive more data.
