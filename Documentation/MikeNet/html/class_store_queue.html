Detailed Description

template<class T>
class StoreQueue< T >

Manages a queue of objects efficiently and safely.

Remarks:
Michael Pryor, 6/28/2010.

The queue manages the objects using pointers, making it very efficient.

Only the queue itself is thread safe, not the use of the objects in the queue.
