Detailed Description

template<class T>
class StoreVector< T >

Manages a vector of objects efficiently and safely.

Remarks:
Michael Pryor, 6/28/2010.

The vector manages the objects using pointers, making it very efficient.

Only the vector itself is thread safe, not the use of the objects in the vector.
