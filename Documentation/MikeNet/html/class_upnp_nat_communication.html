UpnpNatAction* UpnpNatCommunication::natControls [private]

Stores and controls UPnP NAT.

UpnpNatCommunication::actionThread makes changes to NAT whilst this object can manually read from the object but not write to it. This object instructs the thread to make any changes necassary.
