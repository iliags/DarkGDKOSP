Detailed Description

TCP mode where the end of a packet is indicated by a postfix.

Remarks:
Michael Pryor, 6/28/2010.

The end of the packet is indicated by a postfix, which can be of any size and be any combination of bytes.
Packets sent have this postfix appended automatically and packets received are expected to have this postfix.

This class is thread safe.
