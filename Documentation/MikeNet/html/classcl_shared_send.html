Detailed Description

Base class for sendable objects which can be used in conjunction with WSASend.

Remarks:
Michael Pryor, 6/28/2010.

This class reduces the complexity of interacting with winsock's send mechanisms.

This class inherits clCriticalSection because: after this object is added to the send cleanup list, we must have control of this object until finished using all of our own resources, because otherwise they may be cleaned up before we are done using them.
