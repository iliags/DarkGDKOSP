Detailed Description

simple socket with basic functionality

Remarks:
Michael Pryor, 3/28/2010.

The socket wraps around a winsock SOCKET object and is initially unusable.
First the socket needs to be setup, this is done either by WSAAccept when a new TCP connection is made, or more commonly by the SetupSocket method. Once setup socket options can be changed and then finally, the socket is bound to a local address using BindSocket and the socket is ready to be used.

This class is not thread safe.
