Detailed Description

Listening socket used to listen and accept new TCP connections.

Remarks:
Michael Pryor, 6/28/2010.

Accepts or rejects incoming TCP connection attempts. Also stores a template socket object to be used with new clients.

Parameters:

nagleDisabled if true then the nagle algorithm will be disabled
[in] clientSocketTemplate client socket template which describes how clients (who's connections are accepted by this listening socket) should deal with data received and how big their receive buffers should be.
A copy of this object is passed to each client during server instance setup.
The object is not directly used by clSocketListening and so this class will not clean it up.
