Detailed Description

Socket used for UDP data transfer.

Remarks:
Michael Pryor, 3/28/2010.

This class provides functionality specific to the UDP protocol.

This class is not inherently thread safe. Send and RawSend are thread safe.

bool clSocketUDP::Recv (  )  [virtual]

Starts a receive operation via UDP
If WSARecvFrom is successful the result of the operation (which will probably not complete instantly) will be passed to the completion port
Upon completion:

  • recvBuffer will be filled with received data
  • bytes will contain the number of bytes received
  • recvAddr will contain the address that the packet was sent from
    If WSARecvFrom is unsuccessful the operation will not complete so the completion port will receive no notification
    Thus errors must be dealt with straight away by the Recv method
Returns:
true if an error occurred, false if no error occurred
