\subsection{Detailed Description}
Stores a single address with IP and port. \begin{DoxyRemark}{Remarks}
Michael Pryor, 6/28/2010.
\end{DoxyRemark}
Interacts with sockaddr\_\-in winsock structure making use of IP and port easier.\par
\par


This class is thread safe.
