\subsection{Detailed Description}
Manages a group of instances. \begin{DoxyRemark}{Remarks}
Michael Pryor, 6/5/2010.
\end{DoxyRemark}
This instance is used for managing a collection of instances. These instances can be identified by ID (unique only within this instance group).\par
\par


This class is thread safe; however, instances can only be 'finished' by the main process.
