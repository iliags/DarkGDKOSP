Empties the received TCP packet queue. 

When TCP packets are received and a receive function has not been set (in the constructor), packets are put into a queue and received using GetPacketFromStoreTCP. If the queue gets too big newer packets may not be received for some time and so it can be useful to empty the queue. This command discards everything in the queue.\par
\par


Has no impact on the partial packet store. \par
\par
