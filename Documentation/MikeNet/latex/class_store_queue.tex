\subsection{Detailed Description}
\subsubsection*{template$<$class T$>$ class StoreQueue$<$ T $>$}

Manages a queue of objects efficiently and safely. \begin{DoxyRemark}{Remarks}
Michael Pryor, 6/28/2010.
\end{DoxyRemark}
The queue manages the objects using pointers, making it very efficient.\par
\par


Only the queue itself is thread safe, not the use of the objects in the queue.
