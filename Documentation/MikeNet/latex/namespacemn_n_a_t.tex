\subsection{Detailed Description}
Commands for procedural programming, that wrap around internal classes. These commands exist for those who prefer procedural programming over object orientated. They wrap around the classes of the Universal Plug and Play Network Address Translation module and deal with errors differently dependent on the error mode we are in.
